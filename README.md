# git-practice
this is a repository for all git commands practice

1.git init = initializing a repository

2.git clone = download the repository

3.git add = add file to staging area

4.git commit = records file in the version history

5.git push = commits are pushes to remote repository

6.git log = used to list the version history

7.git status = list all the files that to be committed

8.git checkout = used to switch from one branch to another branch

9.git checkout -b = creates new branch and switch to that branch

10.git tag = give tags to specific command

11.git diff = shows the file differences which are not yet staged

12.git reset = undo the file from staging area

12.git fetch = fetch all the latest commits

13.git pull = fetches and merges changes on the remote server to working area
