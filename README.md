# git-practice
this is a repository for all git commands practice
git init = initializing a repository
git clone = download the repository
git add = add file to staging area
git commit = records file in the version history
git push = commits are pushes to remote repository
git log = used to list the version history
git status = list all the files that to be committed
git checkout = used to switch from one branch to another branch
git checkout -b = creates new branch and switch to that branch
git tag = give tags to specific command
git diff = shows the file diferences which are not yet staged
git fetch = fetch all the latest commits
git pull = fetches and merges changes on the remote server to working area
